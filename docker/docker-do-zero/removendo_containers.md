# Removendo containers

docker stop **<ID_CONTAINER>** -> Para a execução do container.
docker start **<ID_CONTAINER>** -> Inicia a execução do container.
docker rm **<ID_CONTAINER>** -> Remove o container

1. Quando um container está em execução, podemos passar um parâmetro para "forçar" a remoção.
   1. docker rm **<ID_CONTAINER>** -f
